# HTML & CSS Mini-Task (Task 6)

This repository contains HTML and CSS files for a mini-task designed to demonstrate the usage of basic HTML tags, form creation, table design, and element manipulation. The project includes solutions to 11 tasks (Q1 to Q11) focusing on HTML structure, debugging, and styling without CSS where specified.

## Project Structure

- **Question1.html**: Fixes bugs in the provided HTML snippet (Q1).
- **Question2.html**: Fixes bugs in a second provided HTML snippet (Q2).
- **Question3.html**: Designs a "Contact Us" form with all fields required (Q3).
- **styleQ3.css**: Styles for the "Contact Us" form in Q3.
- **Question4.html**: Displays a nested list of programming languages, frameworks, and databases (Q4).
- **Question5.html**: Creates a link to open `https://google.com` in a new tab (Q5).
- **Question6.html**: Adds radio buttons for employee type (Salaried and Own Business) to a form (Q6).
- **Question7.html**: Designs a form based on the provided image (Q7).
- **styleQ7.css**: Styles for the form in Q7.
- **Question8.html**: Designs a table based on a provided image (Q8).
- **Question9.html**: Shows default values for all form elements using HTML input tags (Q9).
- **Question10.html**: Highlights the text "HTML & CSS is awesome" without using CSS (Q10).
- **Question11.html**: Creates a form with all types of input elements (Q11).
- **1.png, 2.png, 3.png, 4.png, 5.a.png, 5.b.png, 6.png, 7.png, 8.png, 9.png, 10.png, 11.a.png, 11.b.png**: Screenshots of the code implementations for each task.

## Tasks Overview

### Q1: Fix Bugs in HTML Snippet 1
- Fixes structural errors in a given HTML snippet to ensure proper nesting and tag closure.

### Q2: Fix Bugs in HTML Snippet 2
- Fixes structural errors in a second given HTML snippet to ensure proper nesting and tag closure.

### Q3: Contact Us Form
- Designs a "Contact Us" form with all fields marked as required, styled using `styleQ3.css`.

### Q4: Display Programming Languages and Databases
- Uses HTML list elements to display a nested structure of:
  - Programming Languages (JavaScript, Python, Java) with their frameworks.
  - Databases (MySQL, MongoDB, Cassandra).

### Q5: Link to Open Google in New Tab
- Creates an HTML element (anchor tag) that opens `https://google.com` in a new tab.

### Q6: Radio Buttons for Employee Type
- Adds two radio buttons to a form for employee type (Salaried and Own Business), grouped together.

### Q7: Form Design Based on Image
- Designs a form based on the provided image (http://evc-cit.info/cit040/formguide/card_0.png), styled using `styleQ7.css`.

### Q8: Table Design Based on Image
- Uses the `<table>` tag to design a table based on a provided image.

### Q9: Default Values for Form Elements
- Creates HTML input tags with default values for various form elements.

### Q10: Highlight Text Without CSS
- Highlights the text "HTML & CSS is awesome" using HTML elements (e.g., `<mark>`) without CSS.

### Q11: Form with All Input Types
- Creates an HTML page containing all types of input elements (e.g., text, password, radio, checkbox, etc.).

1. **Clone the Repository**:
   ```bash

   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd task-6
   ```
3. **Open the HTML Files**:
   - Open each `QuestionX.html` file in a web browser to view the solution.
   - For Q3 and Q7, ensure the corresponding CSS files (`styleQ3.css` and `styleQ7.css`) are in the same directory.
